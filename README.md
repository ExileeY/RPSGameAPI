# Rock-Paper-Scissors game API

## How to use
1. Clone repository
```
git clone git@github.com:ExileeY/RPSGameAPI.git
```
2. Navigate to the cloned repository
```
cd RPSGameAPI
```
3. Install dependencies
```
bundle install
```
4. Run migrations
```
rails db:migrate
```
5. Run application
```
rails server
```
